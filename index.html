Bienvenido a PyEgar V0.5

PyEgar es un motor 2D diseñado para la velocidad de desarrollo. En la V0.5, introducimos una comunicación más profunda entre componentes mediante el bus de estado unificado.

Filosofía de Inyección

En lugar de usar herencia compleja, PyEgar utiliza libs(self, ...). Esto inyecta métodos directamente en tu objeto, permitiendo que un "Jugador" tenga capacidades físicas, visuales y de cámara sin jerarquías rígidas.

Inicio Rápido

Así luce una entidad básica en la V0.5:

from pyegar import *

class Heroe:
    def __init__(self):
        # Inyectamos componentes inteligentes
        libs(self, CharacterBody2D, Sprite2D, Camera2D)
        
        self.sprite.load_texture("hero.png")
        self.camera.follow(self) # La cámara me sigue a mí
        self.eg_Speed(300)

    def on_update(self):
        self.mov() # Movimiento con Delta Time incluido

# Iniciar motor
game = win(init())
game.start()

Estado del Proyecto

Novedades V0.5

  • Delta Time Nativo: Movimiento independiente de los FPS.
  • DrawingComponent: Dibujo vectorial auto-posicionado.
  • Camera Follow: Seguimiento suavizado de objetivos.

Limitaciones

  • AABB: Las colisiones no soportan rotación avanzada.
  • Pygame: Sigue siendo la base del backend de renderizado.
