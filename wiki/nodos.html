API de Nodos

En la V0.5, los nodos son componentes inteligentes. Haz clic en los títulos para ver la documentación detallada de cada uno.

CharacterBody2D

Core / Cinemática

Nodo base para entidades con movimiento. Soporta Delta Time nativo.

Sprite2D

Gráficos / Texturas

Gestiona la visualización de imágenes y el auto-volteo según dirección.

DrawingComponent

Gráficos / Vectorial

Permite dibujar formas geométricas (rect, circle) que siguen al nodo automáticamente.

CollisionShape2D

Física / Eventos

Gestiona hitboxes y detección de mouse con soporte para scroll de cámara.

Camera2D

Viewport

Controla el desplazamiento de la vista y el seguimiento suavizado de objetivos.
