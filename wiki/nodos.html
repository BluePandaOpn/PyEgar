API de Nodos

Aquí encontrarás la lista de componentes inyectables que componen el motor. Cada nodo añade atributos y métodos únicos al objeto mediante la función libs().

CharacterBody2D

Física / Movimiento

Es el nodo base para cualquier entidad que requiera coordenadas (x, y) y movimiento controlado por velocidad.

Métodos Públicos

  • eg_Speed(float): Establece la velocidad en píxeles/segundo.
  • mov(): Habilita el listener de teclado (WASD/Flechas).

Ejemplo de uso práctico:

class Player:
    def __init__(self):
        libs(self, CharacterBody2D)
        self.eg_Speed(200) # px/s
        self.mov()         # Movimiento automático

Sprite2D

Gráficos

Gestiona la visualización de imágenes y el auto-volteo (flip) según la dirección de movimiento.

Métodos Públicos

  • load_texture(path, w, h): Carga un archivo PNG/JPG.
  • flip(bool): Voltea la imagen manualmente en el eje X.

Ejemplo con CharacterBody:

class Enemigo:
    def __init__(self):
        libs(self, CharacterBody2D, Sprite2D)
        self.load_texture("assets/enemy.png", 64, 64)
        # Nota: El sprite mirará a la izquierda automáticamente 
        # si x_direction es negativa.

CollisionShape2D

Interacción

Añade una caja de colisión (Hitbox) y permite interactuar con el mouse.

Métodos Públicos

  • rect(w, h, color): Define el tamaño y color del borde.
  • set_drag(True): Permite arrastrar el objeto con el ratón.
  • is_colliding_with(otro): Verifica contacto con otra hitbox.

Detección de colisiones:

def update(self):
    if self.eg_collision.is_colliding_with(moneda.eg_collision):
        print("¡Moneda recogida!")

Label

UI / Interfaz

Sistema de renderizado de texto optimizado. Utiliza un caché de fuentes compartido para no consumir CPU extra.

Métodos Públicos

  • printf(text, size, color, x, y, font): Dibuja texto en pantalla.

Interfaz de usuario:

class HUD:
    def __init__(self):
        libs(self, Label)
        
    def on_draw(self):
        self.printf(f"FPS: {engine.get_fps()}", size=20, color=GOLD, x=10, y=10)
