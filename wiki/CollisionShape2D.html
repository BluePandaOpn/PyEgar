CollisionShape2D

Gestión de Hitboxes, Física AABB y Detección de Eventos.

¿Qué es CollisionShape2D?

Es el componente que otorga "presencia física" a un nodo. Permite que el Kernel detecte cuando dos objetos se tocan o cuando el puntero del mouse interactúa con una entidad.

Interacción con el Mouse

Una de las funciones más potentes en la V0.5 es is_hover(). A diferencia de otros motores, PyEgar calcula la posición del mouse relativa a la Camera2D automáticamente.

if self.collision.is_hover(): # El mouse está encima

Métodos Principales

rect(w, h)

Define un área rectangular de colisión. Se sincroniza con el Pipeline de Render para mantenerse sobre el Sprite.

is_colliding_with(obj)

Devuelve True si la hitbox actual solapa con la del objeto objetivo (Detección AABB).

Implementación Práctica

Para crear un objeto clickeable (como una carta en el Memory Game):

class Carta:
    def __init__(self):
        libs(self, Sprite2D, CollisionShape2D)
        self.collision.rect(100, 150) # Tamaño de la hitbox

    def on_update(self):
        if self.collision.is_hover() and mouse.left_click():
            self.voltear_carta()
