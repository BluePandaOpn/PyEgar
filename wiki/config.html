Configuración Global

PyEgar elimina la necesidad de archivos .json o .xml complejos. El motor utiliza reflexión de código para leer variables globales directamente desde tu script principal.

Parámetros de Arranque

Define estas variables en la parte superior de tu main.py para que el motor las reconozca automáticamente:

Ancho de Ventana

Define el tamaño horizontal en píxeles.

WIDTH = 1280

Alto de Ventana

Define el tamaño vertical en píxeles.

HEIGHT = 720

Título

Texto que aparecerá en la barra superior.

TITLE = "Mi Gran Juego"

Frames por Segundo

Controla la tasa de refresco (FPS).

FPS = 60

Cómo funciona internamente

Cuando llamas a init() en Wind.py, el motor ejecuta un escaneo del módulo __main__:

Proceso de Carga:

  1. El Kernel importa el módulo __main__.
  2. Busca los atributos mediante getattr().
  3. Si no encuentra una variable (ej. TITLE), aplica un valor por defecto (Fallback).
  4. Inicializa el contexto de pygame.display con esos datos.
# Ejemplo de main.py configurado
from pyegar import *

WIDTH = 1000
HEIGHT = 500
TITLE = "Runner 2D"
FPS = 120

# El motor leerá automáticamente las variables de arriba
init()

Valores por Defecto

Si omites alguna configuración, ERROREGAS y el Kernel aplicarán estos valores de seguridad para evitar que el juego falle:

Variable Valor Fallback
WIDTH / HEIGHT 800 x 600
TITLE "PyEgar Engine"
FPS 60
