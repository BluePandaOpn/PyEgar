DrawingComponent (eg_draw)

El sistema de dibujo vectorial nativo del Kernel V0.5.

¿Qué es eg_draw?

Es un componente inyectable que permite a cualquier nodo dibujar formas geométricas directamente en la pantalla sin necesidad de cargar texturas externas. Es ideal para prototipado rápido, interfaces y efectos visuales dinámicos.

Rectángulos

Dibuja rectángulos con bordes redondeados o sólidos.

self.eg_draw.rect(width, height, color)

Círculos

Dibuja círculos perfectos centrados en el nodo.

self.eg_draw.circle(radius, color)

Ejemplo de Implementación

Para usarlo, simplemente inyéctalo en el libs() y llámalo dentro de on_draw().

from pyegar import *

class CajaMagica:
    def __init__(self):
        # Inyectamos el componente de dibujo
        libs(self, DrawingComponent)
        self.x, self.y = 100, 100

    def on_draw(self):
        # Dibujamos un cuadro azul de 50x50
        self.eg_draw.rect(50, 50, BLUE)

# En el main
render(CajaMagica())

Sincronización Automática

Todas las formas dibujadas con eg_draw se desplazan automáticamente con la Camera2D y respetan el Z-Index del nodo.
